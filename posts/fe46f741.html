C++数组和字符串

发表于 2019-09-07 | 更新于: 2019-09-07 | 分类于 C++ | 阅读数
字数统计: 1.5k | 阅读时长 ≈ 6

数组：

定义数组时，[]最好是常量

int n = 10;
int a[n]; //VS编译不过

define Size 10

int a[Size];

使用数组时，下标可以是常量，可以是变量，也可以是表达式

数组初始化

1.数组的初始化需要{}，只有在定义时才能初始化
int a1[3] = {1,2,3};

int a2[3];
a2={1,2,3} //err
//数组明不是变量，它是常量，常量不能修改

2.部分初始化，其它自动初始化为0
int a3[10] = {1,2,3}；
0~2（前三个元素）分别为1,2,3，其它初始化为0

3.数组元素全部初始化为0
int a4[10]={0};

4.如果定义时初始化，第一个[]内部可以不写内容
int a5[] = {1,2,3,4,5,6,7,8,9};
编译器会根据用户初始化的元素来确定数组大小

int　a6[]; //err

数组不初始化，它的值为随机数

  1. 数组名是常量不能修改
  2. 数组名是数组首元素地址
  3. sizeof(数组名)测数组总大小
  4. 元素个数：中大小/某个元素大小

二维数组：

a[3][4]

  • 内存中没有多维，都只有一维，多维数组是特殊的一维数组。
  • 定义了一个一维数组a[4]，这个一维数组有三个元素，每个元素int[4]
  • a[0],a[1],a[2]就是第0，1，2元素的数组名
  • 二维数组可以理解为m行n列，没有a[3][4]此元素

求行数和列数

a[5][10];

  • 求行数：中大小/每行的大小 = sizeof(a)/sizeof(a[0]);
  • 求列数:n = sizeof(a[0])/sizeof(int);//默认int类型或者直接 sizeof(a[0])/sizeof(a[0][0]);
  • 行*列(总元素个数)：n = sizeof(a)/sizeof(a[0][0]);

字符串

  1. C语言没有字符串类型，用字符数组模拟
  2. 字符串一定是字符数组，字符数组就不一定是字符串
  3. 如果字符数组以字符’\0’(‘\0’等于数字0)结尾，那么这个字符数组就是字符串
    char b[]={‘a’,’b’,’c’};//字符数组
    char c[10]={‘a’,’b’,’c’,’\0’};字符串
    char d[10]={‘a’,’b’,’c’,0};//字符串

char a1[]={‘a’,’b’,’c’};
printf(“%s”,a1);//输出乱码，因为没有结束符

char a1[]={‘a’,’b’,’c’,0};
printf(“%s”,a1);//输出abc

char a[10] = “abc”；//常用的初始化，在字符串结尾自动加结束符

char a[]=”\012abc”;
这里的\012可能组成一个转义字符,最好不要加数字

char a[]=”\0abc” 不会输出abc，因为在之前已经结束

char a[10]=”123456789”最多写9个字符，留一个位置放结束符

sizeof()测数据类型大小，不会因为结束符提前结束.

char strcpy(char des,char src);

char strcat(const char str1,const char str2);
char strncat(const char str1,const char str2,int size);//字符串连接函数，将str2指向的字符串连接到str1指向的字符后面，同时会删除str1后面的’\0’,返回的是str1指向字符串的首地址重点内容
